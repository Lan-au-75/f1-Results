# Cài Đặt

1.Chạy dự án bằng lệnh sau

Sử dụng node v18.16.0 lts

```
git clone https://github.com/Lan-au-75/f1-Results.git
or
git clone git@github.com:Lan-au-75/f1-Results.git

npm i

npm start

npm run json (lấy api)

```
